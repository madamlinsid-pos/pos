Backup & Sync
By default this app only stores data in this browser. Connect Google Drive so Owner, Branch 1 and Branch 2 stay in sync automatically.
Google Drive Sync
Sign in with the same Google account on every device (Owner, Branch 1, Branch 2) — that's what lets them all reach the same data file. Once connected, changes sync automatically about every 15 seconds, and right after you make an edit.
Need a Client ID? In Google Cloud Console, create a project → enable the Google Drive API → create an OAuth Client ID (type: Web application) → add the URL where this file is hosted under "Authorized JavaScript origins". Google sign-in requires this file to be hosted at an http/https address — it won't work opened directly as a local file.
